# zoomitlist

This project created for listing the latest posts of [zoomit](https://zoomit.ir) site in different way or even better.

(The spiritual value of all posts and content belongs to Zommit site)

For watching the live version see [this](https://zoomitlist.herokuapp.com)
## Run the project
For running the project you should have node.js installed and after clone the project, in project's directory run :
```bash
npm ci
```
or
```bash
npm install
```
